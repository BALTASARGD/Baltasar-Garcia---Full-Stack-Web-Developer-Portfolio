# **App Name**: Baltasar Garcia - Full Stack Web Developer Portfolio

## Core Features:

- Profile Display: Display personal and professional details of Baltasar Garcia.
- Skills Section: Showcase Baltasar's technical skills and language proficiency.
- Experience Timeline: Display Baltasar's professional experience in a timeline format.
- Projects Showcase: Display personal projects with links to GitHub repositories.
- Contact Section: Provide contact details and links to social media profiles.
- Skill Recommendation Tool: AI powered tool for personalized recommendation to improve and acquire skills

## Style Guidelines:

- Primary color: A deep, professional blue (#294B63) to convey trust and stability.
- Background color: A very light desaturated blue-gray (#F0F4F8) to provide a clean backdrop.
- Accent color: A muted teal (#3B828F) for highlights and interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Use minimalist, professional icons for navigation and skill representation.
- Single-page layout with smooth scrolling to each section. The header section to have a full-width background with centered content.
- Subtle fade-in animations on scroll to create a dynamic user experience.
